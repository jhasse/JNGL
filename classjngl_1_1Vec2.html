Detailed Description

Two-dimensional vector.

JNGL doesn't provide any linear algebra functions itself but relies on the excellent Boost.QVM:

#include <jngl/Vec2.hpp>
jngl::Vec2 v(3, 4);
double length = boost::qvm::mag(v);
assert(length == 5);
